プライバシーポリシー

最終更新日：2026年4月9日

1. はじめに

Kagi-Bus（以下「本アプリ」）は、千歳科学技術大学のバス時刻表を表示するアプリです。本ポリシーは、本アプリが収集・利用する情報について説明します。

2. 収集する情報

本アプリが収集・保存する情報は以下のとおりです。

  • 通知設定（有効／無効、通知タイミング）：お使いの端末内にのみ保存され、外部へ送信されることはありません。
  • クラッシュレポート・エラーログ：アプリが異常終了した場合や API エラーが発生した場合に、端末情報（OS バージョン、端末モデル、アプリバージョン）およびエラーの発生状況を自動的に収集します。これらはアプリの安定性向上を目的として Firebase Crashlytics（Google LLC）へ送信されます。

氏名・メールアドレス・位置情報などの個人情報は一切収集しません。

3. 外部通信

本アプリは以下の外部サービスと通信します。

  • Google Apps Script（GAS）API：バス時刻表データの取得に使用します。このリクエストには個人を特定できる情報は含まれません。
  • Firebase Crashlytics（Google LLC）：クラッシュレポート・エラーログの収集に使用します。収集されたデータは Google のプライバシーポリシー（https://policies.google.com/privacy）に従って処理されます。

4. 情報の第三者提供

本アプリはクラッシュレポート・エラーログを Firebase Crashlytics（Google LLC）へ送信します。それ以外の第三者への個人情報の提供は行いません。

5. お問い合わせ

本ポリシーに関するご質問は、以下の GitHub Issues よりお問い合わせください。

https://github.com/myu-mikazuki/Chitose-bus/issues

Privacy Policy

Last updated: April 9, 2026

1. Introduction

Kagi-Bus (the "App") displays bus timetables for Chitose Institute of Science and Technology. This policy describes what information the App collects and how it is used.

2. Information We Collect

The App collects and stores the following information:

  • Notification settings (enabled/disabled, timing preference): stored locally on your device only and never transmitted externally.
  • Crash reports and error logs: when the App crashes or an API error occurs, device information (OS version, device model, app version) and error details are automatically collected. This data is sent to Firebase Crashlytics (Google LLC) for the purpose of improving app stability.

We do not collect any personal information such as name, email address, or location data.

3. External Communication

The App communicates with the following external services:

  • Google Apps Script (GAS) API: used to retrieve bus timetable data. These requests do not include any personally identifiable information.
  • Firebase Crashlytics (Google LLC): used to collect crash reports and error logs. Data collected is processed in accordance with Google's Privacy Policy (https://policies.google.com/privacy).

4. Sharing of Information

The App sends crash reports and error logs to Firebase Crashlytics (Google LLC). No other personal data is shared with third parties.

5. Contact

If you have any questions about this policy, please contact us via GitHub Issues:

https://github.com/myu-mikazuki/Chitose-bus/issues
