プライバシーポリシー

最終更新日：2026年3月27日

1. はじめに

Kagi-Bus（以下「本アプリ」）は、千歳科学技術大学のバス時刻表を表示するアプリです。本ポリシーは、本アプリが収集・利用する情報について説明します。

2. 収集する情報

本アプリが収集・保存する情報は以下のみです。

  • 通知設定（有効／無効、通知タイミング）：お使いの端末内にのみ保存され、外部へ送信されることはありません。

氏名・メールアドレス・位置情報などの個人情報は一切収集しません。

3. 外部通信

本アプリはバス時刻表データを取得するため、Google Apps Script（GAS）が提供する API にアクセスします。このリクエストには個人を特定できる情報は含まれません。

4. 個人情報の第三者提供

本アプリは個人情報を収集しないため、第三者への提供は行いません。

5. お問い合わせ

本ポリシーに関するご質問は、以下の GitHub Issues よりお問い合わせください。

https://github.com/Yuzucchi-cist/Chitose-bus/issues

Privacy Policy

Last updated: March 27, 2026

1. Introduction

Kagi-Bus (the "App") displays bus timetables for Chitose Institute of Science and Technology. This policy describes what information the App collects and how it is used.

2. Information We Collect

The App collects and stores only the following information:

  • Notification settings (enabled/disabled, timing preference): stored locally on your device only and never transmitted externally.

We do not collect any personal information such as name, email address, or location data.

3. External Communication

The App accesses an API provided by Google Apps Script (GAS) to retrieve bus timetable data. These requests do not include any personally identifiable information.

4. Sharing of Information

Because the App does not collect personal information, no personal data is shared with third parties.

5. Contact

If you have any questions about this policy, please contact us via GitHub Issues:

https://github.com/Yuzucchi-cist/Chitose-bus/issues
